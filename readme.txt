== Description ==

Simple plurk sharing button with counter.

== Installation ==

1. Download the plugin and extract its contents.
2. Upload the `ricki-plurk-count` folder to the `/wp-content/plugins/` directory.
3. Activate **Ricki Plurk Count** plugin through the 'Plugins' menu in WordPress.

That's it!
